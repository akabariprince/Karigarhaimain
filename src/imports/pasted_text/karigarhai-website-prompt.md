### GLOBAL DESIGN RULES

**Responsive breakpoints**:
- Mobile: 320px – 480px
- Tablet: 481px – 1024px
- Laptop: 1025px – 1440px
- Large screen: 1441px+

**Every section must have**:
- Proper heading hierarchy (H1 → H2 → H3)
- CTA button (at least one per section)
- Responsive layout shift from desktop to mobile
- Lazy loaded images
- Accessible alt text on all images

**Animation guidelines**:
- Scroll-triggered fade-in on sections (subtle, 0.3s)
- Counter animation on stats numbers
- Smooth tab/toggle transitions
- Hover states on all cards and buttons
- No heavy animations that slow page speed

**SEO requirements**:
- Meta title + description per page
- OG image per page
- Schema markup for FAQ sections
- Sitemap.xml
- Robots.txt

**Performance targets**:
- Lighthouse score 90+
- Core Web Vitals: LCP under 2.5s, CLS under 0.1
- All images in WebP format
